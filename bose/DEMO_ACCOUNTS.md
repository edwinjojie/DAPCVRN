# ğŸ¯ **BOSE 4-Role System**

## The 4 Core Roles

BOSE uses a **4-role system**:

1. **Student** - Individual user managing credentials
2. **University** - Authorized issuer/validator of academic credentials
3. **Recruiter** - Authorized verifier and consumer of credentials
4. **Admin** - System administrator
